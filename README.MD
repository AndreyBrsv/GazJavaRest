# Сборка и запуск проекта

- Склонируйте репозиторий
```$xslt
git clone https://github.com/AndreyBrsv/GazJavaRest.git
```
- Сделайте импорт проекта в вашу IDE

- Выполните команду
```shell script
gradle test bootRun
```
